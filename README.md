# DiscordBTTVBot
Very simple discord bot that runs on my computer. I wont be sharing bot but you can see the code

Prefix is 
```
=<
```
# Commands
Add emote
```
=< add EMOTE_NAME EMOTE_LINK
```
Remove emote
```
=< remove EMOTE_NAME
```
Emote list
```
=< emotelist
```
EMOTE_NAME can be anything you want

EMOTE_LINK must be BTTV emote id
